# 🚚 SmartDelivery Tracker (SmartDeliveryFX)

SmartDelivery Tracker, JavaFX ile geliştirilmiş bir kargo takip sistemidir.  
Bu proje, veri yapıları kullanılarak gerçek bir lojistik sürecin basitleştirilmiş simülasyonunu sunar.

---

## 💡 Proje Fikri

Sistem, iki temel veri yapısını birlikte kullanır:

- 📦 **Queue (Kuyruk):** Kargoların işlem sırasını yönetir (FIFO mantığı)
- 🌳 **BST (Binary Search Tree):** Kargoları tracking ID üzerinden hızlı aramak için kullanılır

Bu yapı sayesinde hem **sıralı işlem**, hem de **hızlı arama** yapılabilir.

---

## ✨ Özellikler

- ➕ Kargo ekleme (ID, gönderici, alıcı)
- 📦 Kargo işleme (rastgele teslim süresi ile simülasyon)
- 🔍 Tracking ID ile kargo arama (BST search)
- ⏱ Ortalama teslim süresi hesaplama
- 📋 Teslim edilen kargoların takibi (statü güncelleme)
- 🖥 JavaFX tabanlı basit arayüz

---

## 🧠 Kullanılan Veri Yapıları

### Queue (LinkedList)
- Kargoların geliş sırasını tutar
- FIFO (First In First Out)

### Binary Search Tree (BST)
- Tracking ID bazlı hızlı arama
- Logaritmik arama performansı

---

## 🛠️ Kullanılan Teknolojiler

- Java
- JavaFX
- OOP (Object Oriented Programming)
- Data Structures (Queue, BST)
